# Recipe App

A basic web app using the [recipeapi](https://recipeapi.io/docs/resources/recipes/) to discover recipes by name via a case-insensitive search. Each result is displayed as a clickable card showing the recipe's cuisine, meal type, difficulty, prep and cook times, calories, protein, and dietary tags; clicking a card opens a detailed modal with a full description, per-serving nutrition stats, a categorised ingredient list with quantities, and step-by-step cooking instructions.

Originally used the Edamam API but has been updated with a free API version for recipes with minor changes to accommodate the new API.
